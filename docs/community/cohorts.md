# Cohort program

Structured, time-boxed small-group cohorts running through the
[30 Days of OpenQuant](30-days.md) challenge together.

Full program spec lives at
[`community/cohort-program.md`](https://github.com/revorhq/openquant/blob/main/community/cohort-program.md).

## Format

- **Cohort size:** 15 – 25.
- **Duration:** 5 weeks (4 weeks of building + 1 demo week).
- **Cadence:** two 60-minute calls a week, plus async Discord.
- **Output:** every member ships at least one PR to the project.

## Roles

| Role          | What they do |
| ------------- | ------------ |
| Cohort lead   | Runs weekly calls, triages blockers. |
| Mentors (2-3) | Review PRs, hold office hours. |
| Members       | Build, ship, review each other's PRs. |

## Applying

Cohort applications open quarterly via the Discord. We prioritize:

- Existing Indian-markets context (you know what STT is, you've used Kite).
- Willingness to ship in public.
- At least one prior open source contribution somewhere — doesn't have to be quant.

## Partner channels

- GDG chapters
- Quant clubs at IITs / IIMs / IISc / IIITs
- Algo-trading community Telegrams (selected)

If you run a community and want to host a chapter, file an issue with
the `cohort-partnership` label.
